# Expense API
A secure fastapi application for managing personal expenses with user authentication
## Features
- JWT based authentication
- Expense management
- Protected routes
- database migration managed with Alembic
- PostgreSQL support with SQLALchemy ORM
- Pydantic models for request/response validation

## Tech Stack
- FastAPI
- Python 3.14
- PostgreSQL
- Alembic
- JWT
- Pydantic
- Uvicorn
